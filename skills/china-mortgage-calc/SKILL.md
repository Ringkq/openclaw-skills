# 中国房贷计算器 (china-mortgage-calc)

## 功能概览

1. **等额本息计算** - 每月还款固定，前期利息多
2. **等额本金计算** - 每月还款递减，总利息少
3. **两种方式对比** - 帮你选择最优方案
4. **公积金贷款** - 利率更低，优先使用
5. **组合贷款** - 公积金+商业贷款混合
6. **提前还款分析** - 计算提前还款节省的利息

## 使用场景

- "贷款100万，30年，月供多少？"
- "等额本息和等额本金哪个合算？"
- "公积金能贷多少？利率是多少？"
- "提前还10万，能省多少利息？"
- "现在LPR是多少？"

## 2024年最新利率

- LPR（5年以上）：3.95%
- 公积金贷款利率：3.1%（5年以上）
- 首套房商业贷款：LPR - 0.45% = 3.5%
