# 高考志愿填报智能助手 (gaokao-volunteer)

## 功能概览

帮助高考考生科学填报志愿，提供：

1. **院校推荐** - 根据分数/排名推荐适合的院校
2. **专业分析** - 热门专业就业前景、薪资数据
3. **录取概率** - 基于历年数据分析录取可能性
4. **冲稳保策略** - 科学配置志愿梯度
5. **平行志愿解读** - 各省平行志愿规则说明

## 使用场景

- 考生查询"我考了XXX分，能上什么学校？"
- 家长询问"XX专业就业前景怎么样？"
- 考生需要"帮我制定冲稳保志愿方案"
- 查询"XX大学XX专业近三年录取分数线"

## 数据来源

- 教育部阳光高考平台：https://gaokao.chsi.com.cn/
- 各省招生考试院官网
- 掌上高考 API

## 使用方法

### 1. 查询院校推荐

```
用户：我是湖南考生，理科，考了580分，帮我推荐院校
```

助手会：
1. 调用 `scripts/recommend_schools.py` 查询历年录取数据
2. 按冲/稳/保三档推荐院校
3. 提供各院校近三年录取分数线

### 2. 专业分析

```
用户：计算机科学与技术专业就业怎么样？
```

助手会：
1. 调用 `scripts/analyze_major.py` 获取专业数据
2. 返回就业率、平均薪资、主要就业方向

### 3. 制定志愿方案

```
用户：帮我制定完整的志愿填报方案
```

助手会：
1. 收集考生信息（分数、省份、科目、意向城市/专业）
2. 生成冲稳保志愿表
3. 提供填报建议

## 注意事项

- 数据仅供参考，以各省招生院校官方公布为准
- 录取分数线每年有波动，建议参考近3年数据
- 平行志愿各省规则不同，请以本省招生办规定为准
