# 中国驾考智能助手 (jiakao-helper)

## 功能概览

帮助考生高效备考驾照，提供：

1. **科目一题库** - 交通法规、标志标线、安全驾驶
2. **科目四题库** - 文明驾驶、紧急情况处理
3. **易错题分析** - 高频错题详解
4. **科目二要点** - 倒车入库、侧方停车、坡道定点等
5. **科目三要点** - 起步、变道、超车、路口通行等
6. **考试技巧** - 答题策略、注意事项

## 使用场景

- "帮我出10道科目一练习题"
- "科目二倒车入库怎么操作？"
- "交通标志中黄色标志代表什么？"
- "科目三考试有哪些扣分项？"
- "驾考科目一有哪些易错题？"

## 使用方法

### 练习题目

```
用户：出5道科目一选择题
```

助手会调用 `scripts/quiz.py` 随机出题并解析答案。

### 查询知识点

```
用户：限速标志有哪些？
```

助手会从知识库中检索相关内容并详细解释。

### 科目二/三技巧

```
用户：科目二倒车入库的技巧是什么？
```

助手会提供详细的操作步骤和注意事项。

## 数据来源

- 交通运输部驾驶人考试题库（官方）
- 驾考宝典公开题库
- 各地驾校教学资料整理
