# 中医体质分析助手 (tcm-constitution)

## 功能概览

1. **九种体质辨识** - 判断你属于哪种体质
2. **体质调理方案** - 针对性的饮食和生活建议
3. **食疗方案** - 适合各体质的食物和禁忌
4. **穴位保健** - 常用保健穴位按摩方法
5. **季节养生** - 春夏秋冬四季养生要点

- "我总是怕冷，是什么体质？"
