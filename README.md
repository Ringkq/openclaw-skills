# OpenClaw Skills - 中国本土刚需技能集

专为中国用户开发的 OpenClaw/QClaw 技能包，填补 SkillHub 市场空白。

## 技能列表

| 技能 | 描述 | 目标用户 |
|------|------|----------|
| [stock-ipo-alert](./skills/stock-ipo-alert) | A股打新提醒 - 新股申购日历、中签率查询 | A股投资者 |
| [gaokao-volunteer](./skills/gaokao-volunteer) | 高考志愿填报助手 - 院校推荐、专业分析 | 高考考生 |
| [jiakao-helper](./skills/jiakao-helper) | 驾考题库助手 - 科目一/四题库 | 驾考学员 |
| [china-gov-service](./skills/china-gov-service) | 政务服务助手 - 户口/公积金/社保 | 全体市民 |
| [chronic-disease-helper](./skills/chronic-disease-helper) | 慢性病管理 - 糖尿病/高血压 | 慢性病患者 |
| [hukou-helper](./skills/hukou-helper) | 户口/落户助手 - 各城市落户政策 | 外来务工人员 |
| [elderly-care](./skills/elderly-care) | 养老服务助手 - 养老金/养老机构 | 老年人及家属 |

## 安装方法

```bash
# 安装单个技能
skillhub install stock-ipo-alert

# 或手动下载 .skill 文件后安装
```
