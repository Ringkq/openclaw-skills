## 安装方法

```bash
skillhub install stock-ipo-alert
skillhub install gaokao-volunteer
# ... 等待 SkillHub 官方收录后可直接安装
```

## GitHub 安装（当前）

```bash
git clone https://github.com/Ringkq/openclaw-skills.git
# 将对应技能目录复制到 ~/.qclaw/workspace/skills/
```
